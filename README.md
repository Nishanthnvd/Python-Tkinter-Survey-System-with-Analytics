# Python-Tkinter-Survey-System-with-Analytics
This project is a Python-based Online Learning Experience Survey System developed using Tkinter for the graphical user interface and MySQL for database management. The application allows users to log in securely, submit their online learning experience through a survey form, and store responses in a database.  
The system also includes data analysis and visualization using Matplotlib, helping to understand user feedback through charts and graphical insights. Passwords are securely stored using SHA-256 hashing, ensuring better security for user authentication.

This project demonstrates the integration of GUI development, database connectivity, user authentication, and data analytics in Python, making it useful for learning full-cycle application development.

Key Features

User login and authentication system

Secure password hashing

Survey form for collecting online learning feedback

MySQL database integration

Data visualization using Matplotlib

Simple and interactive Tkinter GUI
